Working with a script

In the top left corner of RStudio, click “File” then “New file” and then “R script”.

Download the file galaxies.Rdata from Minerva and save it in a folder on your computer. If the file is called, for example, galaxies(1).Rdata then please just delete the (1) in the filename.

On my computer, the folder where I save galaxies.Rdata is C:\Users\benjamin\Desktop\R prac, so I set my working directory as:

setwd("C:/Users/benjamin/Desktop/R prac")    # Don't forget the quotation marks.

Click “Run” at the top of the script window to run this code. Notice that the forward slash in the folder location has to be changed to a backward slash in R.

To load the file:

load(file = "galaxies.Rdata")     # Again, don't forget the quotation marks.

Remember to click “Run” for this line of code too, otherwise nothing will happen.

  • GlobularClusters, the number of globular star clusters orbiting the the core of the galaxy.
  • BlackHoleMass, the mass of the central black hole, on a logarithmic scale. A mass of \(6\) means \(10^6\) times the mass of our Sun.
  • Type, the visual appearance: Elliptical, Spiral or Lenticular.

We can even identify specific observation numbers:
